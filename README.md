🌍 Azure Translator Dashboard

A modern web-based language translator built using Microsoft Azure Translator API.
This project allows users to input text and translate it into five different languages simultaneously, with a visually appealing dashboard and flag-based language selection.

🚀 Features
🌐 Translate text into 5 languages at once
🎨 Clean and responsive dashboard UI
🏳️ Language selection with country flags
⚡ Fast translation using Azure API
💻 Simple frontend (HTML, CSS, JavaScript)
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
Microsoft Azure Translator API
